## Transaction Management

- [x] **TXN-01**: User can create a transaction with type (expense/income), amount, date, category, and optional description/notes

- [x] **TXN-02**: User can edit any of their own transactions
  - *Outcome:* Validated via Phase 1 service + Phase 2 edit mode in TransactionSheet
- [x] **TXN-03**: User can delete a transaction
  - *Outcome:* Validated via Phase 1 service + Phase 2 TransactionDeleteDialog
- [x] **TXN-04**: User can view their transaction list filtered by date range, type, and/or category
  - *Outcome:* Validated via Phase 1 getAll + Phase 2 nuqs filter bar
- [x] **TXN-05**: User can search transactions by description or notes text
  - *Outcome:* Validated via Phase 1 search query + Phase 2 debounced search input

- [x] **DASH-02**: User can see income vs. expenses balance overview for a date range
  - *Outcome:* Validated Phase 3 summary endpoint + Phase 4 SectionCards with savings rate
- [x] **DASH-03**: User can see monthly spending trends chart
  - *Outcome:* Validated Phase 3 trends endpoint + Phase 4 MonthlyTrendsChart (Recharts bar, trailing 6 months)
- [x] **DASH-04**: User can see a recent transactions widget on the dashboard
  - *Outcome:* Validated Phase 3 recent endpoint + Phase 4 RecentTransactionsWidget
