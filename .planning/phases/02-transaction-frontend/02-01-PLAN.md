<!-- Backend API contract (from backend/src/modules/transaction/) -->

Backend routes:
- GET    /api/transaction           → { data: Transaction[], total, page, limit, totalPages }
- POST   /api/transaction           → Transaction (201)
- GET    /api/transaction/:id       → Transaction
- PUT    /api/transaction/:id       → Transaction
- DELETE /api/transaction/:id       → Transaction
- POST   /api/transaction/bulk-delete → { count }

Query params (GET /api/transaction):
  page?: number (default 1), limit?: number (default 20, max 100),
  type?: "expense" | "income", categoryId?: string,
  dateFrom?: string (ISO date), dateTo?: string (ISO date),
  search?: string

Create body: { type, amount, transactionDate, description?, notes?, categoryId? }
Update body: same fields, all optional

Transaction response includes:
  { id, userId, categoryId, type, amount (Decimal as string), currency, transactionDate,
    description, notes, source, createdAt, updatedAt,
    category: { id, name, type, icon, color } | null }
