|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | Spoofing | SectionCards, RecentTransactionsWidget | mitigate | Both use `useSuspenseQuery` through Axios with `withCredentials: true` — unauthenticated requests return 401 from backend |
| T-04-05 | Info Disclosure | RecentTransactionsWidget | accept | Transaction data belongs to authenticated user only — backend filters by userId; no cross-user leakage possible from frontend |
| T-04-06 | Tampering (XSS) | Transaction description/notes rendered in widget | mitigate | React escapes all text content by default — all data rendered as text nodes, never as raw HTML; category icon is an emoji string rendered as text |
